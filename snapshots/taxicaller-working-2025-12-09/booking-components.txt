================================================================================
ERROR HANDLING PATTERN
================================================================================

Before (crashes on object errors):
  <p>{error}</p>

After (safe rendering):
  <p>{safeRender(error)}</p>

The safeRender() function handles:
- null / undefined → returns null
- string → returns as-is
- number / boolean → converts to string
- Error instance → returns error.message
- Objects → formats as JSON in <pre> tag
